CSS لغة

CSS :هي اختصار لـ Cascading Style Sheets، وهي لغة تُستخدم لتنسيق وتصميم صفحات الويب. تتيح CSS التحكم في مظهر العناصر مثل الألوان، الأحجام، المواقع، والخطوط، بالإضافة إلى إضافة تأثيرات بصرية وجعل الصفحات متجاوبة مع مختلف أحجام الشاشات

فوائد CSS:

  • توفير الوقت: يمكن كتابة كود CSS في ملف واحد وإعادة استخدامه في عدة صفحات.
  • سرعة التحميل: المتصفح يقوم بتحميل ملف CSS مرة واحدة فقط ويستخدمه لجميع الصفحات.
  • تصميم متجاوب: تجعل الصفحات تتكيف مع أحجام الشاشات المختلفة.
  • إضافة تأثيرات متقدمة: مثل الظلال، الحركات، وتغيير الألوان.

كيفية كتابة CSS:

  • داخلية: باستخدام وسم style داخل ملف HTML
  • خارجية: عبر ربط ملف CSS خارجي باستخدام link
  • ضمنية: باستخدام خاصية style مباشرة داخل العنصر

إصدارات CSS:

  • CSS1: ركز على فصل التصميم عن المحتوى
  • CSS2: أضاف ميزات مثل التحكم بالمواقع والخطوط.
  • CSS3: قدم ميزات متقدمة مثل Media Queries، الحركات، والتأثيرات.
